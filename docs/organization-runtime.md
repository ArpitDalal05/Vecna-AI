## 2. Department Registry

1. **Cognition Core (Research Department)**:
   - *Purpose*: High-level strategy formulation, task decompositions, and code audits.
   - *Manager*: `Synapse-01`
2. **Distributed Memory (Database Department)**:
   - *Purpose*: Semantic embeddings indexing, episodic memory consolidation, and cache pools.
   - *Manager*: `Mem-04`
3. **Hive Consensus (Operations Department)**:
   - *Purpose*: Resolution of decision proposals, multi-agent debates, and audit logging.
   - *Manager*: `Decide-02`
