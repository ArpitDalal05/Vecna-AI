# Feature Flags Specification

Defines environmental toggles:

* `USE_MOCK_DATA`: Selects mock/Supabase repositories.
* `USE_REAL_AI`: Selects simulated/LLM planning.
* `ENABLE_STREAMING`: Selects streaming responses.
* `ENABLE_AI_LOGGING`: Controls telemetry logs.
* `ENABLE_PROVIDER_FALLBACK`: Enables multi-provider routing.
